# Versionado Arduino Sensors (Monit‑Agro)

Esta carpeta conserva dos iteraciones funcionales del sketch para ESP32/Arduino.

## v1.0_Previous
- Control básico y **registro mínimo** por Serial.
- Enfoque en simplicidad para **pruebas y calibración** inicial.
- Estructura directa del lazo principal (loop) para validar sensores y actuadores.

## v1.1_FINAL
- **Documentación y legibilidad** mejoradas en el código.
- Organización de constantes y parámetros para facilitar **ajustes en campo**.
- Señalización por Serial más clara para **diagnóstico**.
- Base lista para integrar **telemetría** y manejo de errores extendido.
